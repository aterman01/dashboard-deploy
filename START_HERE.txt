OPTION 2: Manual Steps (More Control)
──────────────────────────────────────

Step 1: Create GitHub Repository
   • Go to: https://github.com/new
   • Repository name: netflixing-dashboard
   • Make it: PUBLIC
   • Do NOT check any boxes
   • Click "Create repository"

Step 2: Use GitHub Desktop
   • Open GitHub Desktop
   • File → Add Local Repository
   • Browse to: C:\Users\Antho\Desktop\UNITY2025\dashboard-deploy
   • Click "Add Repository"
   • Click "Publish repository"

Step 3: Deploy to Railway
   • Go to: https://railway.app/new
   • Click "Deploy from GitHub repo"
   • Select: netflixing-dashboard
   • Click "Deploy"
   • Wait 2-3 minutes
   • Settings → Generate Domain (if not automatic)
   • Copy your URL!
